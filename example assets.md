# The Format of Assets Created
## New Task
```
{
    'data': {
        'created by' : 'cam_ID',
        'type' : 'task_type',
        'time' : 'creation_time',
        'description' : ' ',
        'listener' : 'custom string'
        }
}
```
## Data Collected
```
{
    'data':{
        'created by' : 'cam_ID',
        'type' : 'object_type',
        'value' : 'associated value',
        'density' : 'density of people in the area over the last hour',
        'listener' : 'custom string'
    }
}
```
